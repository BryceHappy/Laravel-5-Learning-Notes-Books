Laravel 5 學習筆記

主機環境建置

程式寫完後，我們要上線測試時，還是需要一個完整的正式環境可以執行 Laravel 5，這裡會介紹大家如何不使用 Homestead，在正式主機把 Laravel 5 執行環境建置起來

環境

Ubuntu 14.04 LTS

Nginx

MySQL 5.7

PHP 7.0

Redis

Memcached

Let's Encrypt SSL

安裝流程說明

  • 安裝 Nginx
  • 安裝 php 7
  • 安裝 composer
  • 安裝 MySQL 5.7
  • 安裝 Memcached
  • 安裝 Redis
  • 安裝 Let's Encrypt
